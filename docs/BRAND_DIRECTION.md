# Lulusweets — preliminary brand direction

Status: reference only. The website aesthetic has not yet been approved.

## Source

Primary visual reference: [`logo.jpg`](../logo.jpg)

Correct brand name: **Lulusweets**

## What the logo communicates

- Personal and handmade
- Playful bakery/cupcake identity
- Friendly rather than formal
- Whimsical illustration
- Soft pastel color
- Strong black hand-drawn outlines
- A recognizable circular badge/mark

## Initial palette families

- Pastel bakery blue from the circular background
- Soft frosting pink/lilac
- Warm custard yellow from the bird
- Cherry red accent
- Cream/white neutrals
- Black linework and text

Exact accessible color values will be sampled and tested during Phase 3.

## Design implications

The final interface should feel like a clean digital extension of the logo. It should not use the
unapproved burgundy editorial aesthetic from the exploratory PWA.

Potential directions to review with Lucia:

- Use the circular logo as the app/home-screen mark and header identity.
- Use blue as the main canvas or navigational accent.
- Use pink and yellow selectively for categories, highlights, and illustrations.
- Prefer friendly rounded or hand-drawn supporting details while keeping recipe text highly readable.
- Keep recipe pages clean enough for cooking; the brand should frame the content, not overwhelm it.

## Decisions still required

- Typography pairing
- Light blue versus white primary background
- Amount of illustration beyond the logo
- Category color system
- Card versus list presentation
- Recipe photography treatment
- Whether the cupcake/bird mark appears in every screen or only global navigation
